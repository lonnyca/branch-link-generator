# Branch Link Generator

A config-driven tool for generating Branch deep links for Southwest Airlines email campaigns.

## Live Tool
https://branch-link-generator.vercel.app

## What it does
- Generates production and staging Branch links across 7 schemas
- Supports param, nonparam, and search token link types
- Config-driven — add new links, schemas, and environments by editing branchConfig.json only
- No code changes needed for new links

## How to use
1. Select an environment
2. Select a schema
3. Select a link
4. Fill in param values
5. Click Generate
6. Copy the output

## Local development
1. Clone the repo
2. Open in VS Code
3. Click Go Live with Live Server extension
